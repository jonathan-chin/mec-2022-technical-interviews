# Pythagorean Triplet

Given an array of N integers, write a function that returns true if there is a triplet (a, b, c) that satisfies the equation a^2 + b^2 = c^2, otherwise false

example 1:
input: [3, 2, 4, 6, 5]
output: true
explanation: 3^2 + 4^2 = 5^2

example 2:
input: [3, 8, 5]
output: false
